# CodeAlpha Data Redundancy Removal System
This project is a data redundancy removal web application developed as part of the CodeAlpha Cloud Computing Internship.

## Features

* Duplicate data detection
* Real-time input validation
* Interactive web interface
* User-friendly design
* Cloud computing based project

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

## Project Objective
The objective of this project is to create a system that detects and prevents duplicate data entries by checking for existing records in the database before inserting new ones.

## How to Run the Project

1. Install Python
2. Install Flask using:
```
pip install flask
```
3. Run the application using:
```
python app.py
```
4. Open browser and visit:
```
http://127.0.0.1:5000
```
